# Exp-001: Shorts 聯盟行銷

## 假設
每日發布的 YouTube Shorts 有觀看數，在 description 加入相關聯盟連結，
只要有 0.1% 點擊轉換率，就能產生被動收入。

## 方法
1. 在 pixabay-shorts-bot 的 description template 加入聯盟連結區塊
2. 根據名言 mood 動態選擇相關產品連結
3. 使用 link-in-bio 頁面作為 landing page 集中管理

## 聯盟平台
| 平台 | 適合 | 佣金 |
|------|------|------|
| 博客來AP | 心理/哲學書籍 | 2-4% |
| 讀冊生活 | 二手書/新書 | 3% |
| Amazon Associates | 英文書（Rumi/Dumbledore等） | 1-10% |
| eToro/券商 | 金融/投資類影片 | CPA $50-200 |
| 線上課程平台 | 自我成長內容 | 10-30% |

## KPI
- [ ] 聯盟連結曝光次數
- [ ] 點擊次數
- [ ] 轉換次數
- [ ] 總收入

## 實作
聯盟連結區塊會動態加入 description，根據 mood 變化：

**inspirational:** 勵志書籍、自我成長課程
**healing:** 冥想App、心理學書籍、放鬆音樂
