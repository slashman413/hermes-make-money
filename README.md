# hermes-make-money 💰

用 **GitHub Actions + GitHub Pages** 把現有專案變現的實驗室。

## 策略地圖

```
最快變現 ───────────────────────────────→ 長期收益
    │                                         │
    ▼                                         ▼
 聯盟行銷        AdSense        Premium        模板銷售
 (Shorts)      (工具站)      (TWSE訊號)      (SaaS Starter)
    │             │              │                │
    ▼             ▼              ▼                ▼
  7天見效       30天見效       60天見效         隨時可賣
```

## 實驗清單

| # | 實驗 | 方法 | 預估時間 | 狀態 |
|---|------|------|---------|------|
| 001 | Shorts 聯盟行銷 | 每日 Shorts description 加 affiliate link | ⚡ 7天 | 🟡 規劃中 |
| 002 | 工具站 AdSense | 所有 Pages 工具站串 Google AdSense | 🟢 30天 | 🟡 規劃中 |
| 003 | GitHub Sponsors | 建立分層贊助方案+回饋 | 🟢 14天 | 🟡 規劃中 |
| 004 | TWSE Premium 訊號 | 付費 email/Telegram 盤中訊號 | 🟡 60天 | 🟡 規劃中 |
| 005 | SaaS Starter 銷售 | Gumroad/GitHub Marketplace 上架 | 🟢 7天 | 🟡 規劃中 |

## 收益儀表板

各實驗實際收益會自動記錄在 `results/` 目錄。

## 自動化 Skills

當實驗有正向結果（>0 收入、流量顯著成長、使用者轉換），會自動生成結構化 skill markdown 存到 `skills/`。

## 策略原則

1. **免費優先** — 所有工具保持免費，靠廣告/聯盟/捐贈變現
2. **不破壞使用者體驗** — 廣告不擋功能，聯盟連結不欺騙
3. **自動化** — 能讓 GitHub Actions 做的事絕不手動
4. **可衡量** — 每個實驗都有明確的 KPI（收入/點擊/轉換率）
